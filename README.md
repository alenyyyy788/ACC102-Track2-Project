# ACC102 Track2 Data Analysis Project
## US Tech Giants Stock Price & Volume Analysis (2020–2025)

---

## 1. Project Purpose
This project analyzes the annual average stock price and trading volume of five major US technology companies (Apple, Microsoft, Google, Amazon, Tesla) from 2020 to 2025. The target users are individual investors, finance students, and anyone interested in tech stock market performance.

## 2. Data Source
- Data Source: Yahoo Finance
- Access Date: 17 April 2026
- Key Variables: Company, Year, Avg_Stock_Price (USD), Avg_Volume (million)

## 3. Python Methods
- Data construction and management with **pandas**
- Missing value checking and data cleaning
- Grouped descriptive statistics
- Time-series line plots and bar charts with **matplotlib**

## 4. Key Findings
1. Tesla maintains the highest stock price across the period.
2. Apple has the largest average trading volume, showing high market activity.
3. All five tech stocks show a clear upward price trend.
4. Microsoft and Google have stable and strong growth with low volatility.

## 5. How to Run
1. Clone this repository
2. Install required packages: `pandas`, `matplotlib`
3. Open `stock_analysis.ipynb` in Jupyter or VSCode
4. Run all cells to view analysis and visualizations
